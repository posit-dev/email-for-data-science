API Reference

The Email Object

An email object that in a serializable, previewable format, optimized for emails with content generated by data scientists.

Email A serializable, previewable, sendable email object for data science workflows.
Email.write_preview_email Write a preview HTML file with inline attachments embedded.
Email.write_email_message Convert the Email to a Python EmailMessage.
Email.preview_send_email Send a preview of the email to a test recipient.
Email.write_quarto_json Write the Email to Quarto’s output metadata JSON format.

Uploading emails

Converting emails to Emails, at which point they can be previewed, tested, and sent.

quarto_json_to_email Convert a Quarto output metadata JSON file to an Email
mjml_to_email Convert MJML markup to an Email
redmail_to_email Convert a Redmail EmailMessage object to an Email
yagmail_to_email Convert a Yagmail email object to an Email

Sending

Functions to sending emails with different providers. And a special handy one to bypass the intermediate object if you are sending a quarto email.

send_email_with_gmail Send an Email object via Gmail.
send_email_with_smtp Send an Email object via SMTP.
send_email_with_redmail Send an Email object via Redmail.
send_email_with_yagmail Send an Email object via Yagmail.
send_email_with_mailgun Send an Email object via Mailgun.
send_quarto_email_with_gmail Send an email using Gmail with content from a Quarto metadata JSON file.

Utilities

Previews and more

write_email_message_to_file Writes the HTML content of an email message to a file, inlining any images referenced by Content-ID (cid).
